# yaml-cli

Commands example:

```
root@openipc:/# yaml-cli -i /etc/majestic.yaml -g .video0.codec
h264

root@openipc:/# yaml-cli -i /etc/majestic.yaml -s .video0.codec h265

root@openipc:/# yaml-cli -i /etc/majestic.yaml -g .video0.codec
h265

root@openipc:/# yaml-cli -i /etc/majestic.yaml -d .video0.codec
```
